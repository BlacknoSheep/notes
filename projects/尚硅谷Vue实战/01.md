# 一、初始化

## 1. 使用 vite 创建 vue 项目

```bash
pnpm create vite
pnpm install
```

## 2. 安装需要的包

```bash
pnpm install @types/node -D
pnpm install sass -D
pnpm install vite-plugin-svg-icons -D
pnpm install fast-glob -D # vite-plugin-svg-icons 需要
pnpm install -D vite-plugin-cdn-import
pnpm install element-plus
pnpm install pinia
```
